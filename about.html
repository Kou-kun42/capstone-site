Our Printing Solution

Currently Panda Printing only has one printer, however we plan to expand that in the future. Our main printer is an Ender 3 Pro fitted with several modifications that allow for the best quality prints. The printer has a maximum print size of 220mm x 220mm x 250mm. This printer is equipped with a bowden style extruder using low tolerance Capricorn tubing that greatly reduces stringing. It also has an SKR Mini E3 control board with onboard TMC2209 stepper motor drivers that allow for higher precision and faster print speeds. Manually leveling a print bed can be inconsistent and time consuming. This is why our solution uses a BLTouch auto leveling sensor which operates before every print to create the perfect first layer, everytime. Finally, our printer has the latest Hero Me style hotend cooling system that enables far steeper supportless printing angles than the stock setup.
